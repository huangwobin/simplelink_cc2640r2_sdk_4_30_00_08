uint8_t rfc_CMD_RX_TEST_s::bEnaCmd

0: No alternative trigger command
1: CMD_TRIGGER can be used as an alternative trigger

uint8_t rfc_CMD_RX_TEST_s::bEnaFifo

0: Do not enable FIFO in modem, so that received data is not available
1: Enable FIFO in modem – the data must be read out by the application

uint8_t rfc_CMD_RX_TEST_s::bFsOff

0: Keep frequency synth on after command
1: Turn frequency synth off after command

uint8_t rfc_CMD_RX_TEST_s::bNoSync

0: Run sync search as normal for the configured mode
1: Write correlation thresholds to the maximum value to avoid getting sync
