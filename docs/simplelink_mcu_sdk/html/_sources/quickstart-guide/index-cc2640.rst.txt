Quick Start Guide for SimpleLink\ |trade| |DEVICE| SDK
######################################################

This guide serves as a road map for users developing applications
and products using the TI SimpleLink |DEVICE|
wireless MCU platform. Whether a seasoned developer or just getting
started, TI has created a variety of resources to simplify
development on the |DEVICE| platform. These resources will enhance
your experience with the TI SimpleLink |DEVICE| |version| Software
Development Kit (SDK) from the out-of-the-box demo to production.

This guide will walk you through the initial setup and demonstration of
running your first sample application. The guide explains how to install the
|SDK| and lists the various other tools required to get started
with your first application. It will also suggest further resources to explore.
